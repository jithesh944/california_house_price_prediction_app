## A new python virtual environment

To create environment

'''
conda create -p venv 
'''

To activate
"""
conda activate venv
"""

To Deactivate
"""
conda deactivate
"""


Libraries used avaible in requirements.txt

To install 
'''
pip install -r requirements.txt
'''
